## 理论

**占空比**：高电平持续时间占总时间的比值

**频率**：周期的倒数

**电平**：高电平与低电平之间电压差

**周期**：相邻高电平＋低电平持续时间

<mark>占空比越大，高电平输出时间越长，LED 给人眼的感觉就是更亮；占空比变大，LED 就变亮。</mark>

![PWM原理](/images/Pasted-image-20260715085308.png)

- CNT &lt; CCR → 输出电平 1  
- CNT &gt; CCR → 输出电平 0  

（<mark>CCR</mark> 是**自己设定的值**，从而决定占空比）

## CubeMX / 配置要点

1. 占空比计算公式：

$$D=\frac{CCR}{ARR+1}$$

2. PWM 周期计算公式：

$$f_{\mathrm{PWM}}=\frac{\text{定时器时钟}}{(PSC+1)(ARR+1)}$$

![CubeMX配置](/images/Pasted-image-20260715113146.png)

从上往下依次为 PSC、ARR、CCR 的对应配置。

### 实际操作步骤

1. **计算 ARR**：根据实际确定所需 PWM 频率，设置 PSC 为 0（一般设为 0，方便计算），代入公式 2 计算出 ARR。  
2. **根据需要计算 CCR**：CCR 一般在代码中按使用场景确定和变更，在 CubeMX 中可以先不设置（示例代码中为 `pwmVal`）。

## 代码控制呼吸灯

初始化（`user code 2`）：

```c
HAL_TIM_PWM_Start(&htim3, TIM_CHANNEL_1);
```

控制 LED：

```c
while (1)
{
  if (pwmVal < 500)
  {
    pwmVal++;
    __HAL_TIM_SetCompare(&htim3, TIM_CHANNEL_1, pwmVal);
    Delay(1);
  }
  if (pwmVal > 0)
  {
    pwmVal--;
    __HAL_TIM_SetCompare(&htim3, TIM_CHANNEL_1, pwmVal);
    Delay(1);
  }
  Delay(200);
}
```

`__HAL_TIM_SetCompare(&htim3, TIM_CHANNEL_1, pwmVal)` 的功能相当于设置 `CCR = pwmVal`。  
由公式 1 可知：通过控制 CCR 变大，提高占空比，从而使 LED 变亮。
